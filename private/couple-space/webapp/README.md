# CoupleSpace

A shared space for couples to plan, remember, and grow together.

## Features

- 📸 **Photo Wall** - Upload and share memories together
- 📝 **Daily Notes** - Leave sticky notes for each other
- 📅 **Shared Calendar** - Plan events with color-coded categories
- ✅ **Lists** - Movies, restaurants, groceries, bucket lists, and more
- 🎲 **Date Night Picker** - Random date idea generator with spin wheel
- 🎁 **Gift Wishlists** - Keep track of what you both want
- 🍳 **Recipes** - Save recipes and track what you've cooked together

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env.local
```

3. Add your Supabase credentials to `.env.local`

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000)
